# Hospital Management System REST API

A backend RESTful API built using Spring Boot for managing hospital operations such as patient records, doctor information, appointments, and medical data.

## Features
- CRUD operations for patients
- REST API development following industry standards
- DTO and Entity mapping
- Spring Data JPA integration
- Custom JPQL queries
- Exception handling
- Validation using Bean Validation
- Layered architecture (Controller, Service, Repository)
- Transaction management using @Transactional

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- JPQL
- Maven
- MySQL/PostgreSQL
- Lombok

## Concepts Implemented
- RESTful API Design
- Dependency Injection
- Entity Relationships
- DTO Pattern
- JPQL Queries
- Hibernate ORM
- Transaction Management
- Exception Handling
- Validation

## Project Objective
The goal of this project is to demonstrate backend development skills using Spring Boot, JPA, Hibernate, and JPQL while building a scalable Hospital Management System API.
